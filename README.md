# motheronthepea.be

Website of Mother on the Pea.

## Run locally

To start a local web server hosting the website.

```bash
make install
make serve
```

## Convert SASS files to CSS

```bash
npm run sass
```

To watch for SASS file changes during development:

```bash
npm run sass-watch
```
